# Todo-back-django-restframework

## feathers 
multiple endpoints for user and todo app.
each user has his own blank page to start with. 

### Main Routs 

[GET,POST,PUT,DELETE]
```
admin/
user/
todos/
```

### Sub routs

```
user/register
user/login
user/logout
```

```
todos/task-list/
todos/task-detail/<int:id>
todos/task-update/<int:id>
todos/task-create/
todos/task-delete
```
